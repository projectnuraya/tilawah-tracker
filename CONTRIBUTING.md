## How to Contribute

1.  **Fork the Repo**: Click the "Fork" button at the top right of this page.
2.  **Clone it**: `git clone https://github.com/YOUR_USERNAME/tilawah-tracker.git`
3.  **Create a Branch**: `git checkout -b feature/AmazingFeature`
4.  **Make your changes**.
5.  **Commit**: `git commit -m 'Add some AmazingFeature'`
6.  **Push**: `git push origin feature/AmazingFeature`
7.  **Open a Pull Request**.

## Code Style

- We use **TypeScript** and **Next.js**.
- Please keep code clean and readable.
- If you change the database schema, make sure to include the migration.

## Need Help?

If you have questions, feel free to open an issue or ask in the discussions!
